
# Curso - Blog do seu Negócio parte 1: Estratégias para geração de lead e entregas de valor. Alura - 8h - 26/03/2021
  

O Blog ajuda a criar um relacionamento mais claro e objetivo com os clientes.

Tópico - Sobre (é geralmente a página mais visitada): o texto deve ser claro e conciso e para ajudar a escrever, responda essas perguntas - porque que é importante falar sobre seu negócio? Para quem você escreve esse texto? Quais os benefícios e o que a pessoa vai encontrar no seu blog?

Fazer pesquisa de mercado: verificar a palavra chave mais utilizada no ramo do negócio, e verificar os grupos do Facebook. analisar os assuntos mais abordados, fazer uma lista desses assuntos e a partir daí elaborar os assuntos dos posts, e os menus que vão ficar no Blog.

1.  Uma pesquisa de palavras-chave em grupos do Facebook bem ativos, com mais de 5.000 participantes, observando as perguntas e comentários;
    
2.  Selecionar as 20 palavras que mais apareciam nas dúvidas e comentários, criando uma Lista de Palavras-chave;
    
3.  Listar e visitar os blogs dos 10 principais concorrentes pra comparar as palavras-chave encontradas no Facebook com as usadas pelos demais bancos digitais;
    
4.  Criar uma lista, organizando as palavras pela relevância, ou seja, pela maior frequência de aparição nos textos.

Utilizar também a ferramenta gratuita do Google, o  ["Planejador de Palavras-chave"](https://ads.google.com/intl/pt-BR_br/home/tools/keyword-planner/). para comparar os termos encontrados com os termos que o Planejador indicou.

Criar a página no painel do Wordpress > e depois ir em personalizar a página > (pois tem q associar a página no menu, ela fica oculta) > Menus > Menu do topo.

"Títulos vencedores para montar"
"Palavras super poderosas" - prende a atenção das pessoas (Digital Marketer)
O titulo prende a atenção e o 1° parágrafo tem que ser a parte mais importante depois do titulo, pois vai manter o leitor lendo o texto.
**A= atenção:** usar palavras-chave, palavras super poderosas, nome do meu negócio.
**I= interesse:** trazer temas que prendam o interesse do leitor e a solução do seu produto ao final.
**D= desejo:** entregar o resultado esperado e descrever mais detalhadamente de como a solução que você apresentou pode ajudar a conquistar o objetivo.
 **A= ação/atitude:** ex: clique no link, inscreva-se no nosso programa de milhagem, outra ação diferente. Colocar apenas 1 informação.
 
Colocar no final da postagem o link direcionando para o conteúdo que foi pesquisado.
Destacar as palavras poderosas em negrito.

**Inserir formulário de opt-in:**

Instalar plugin do Mailchimp para wordpress e ativar. Ir em personalizar a página > Widgets > Primary Sidebar > Adicionar Widget > Primary Sidebar > Adicionar Widget. Deve-se abrir uma conta no Mailchimp para ativar o Widget.

Mailchimp > Conta > Extras > API Keys > gerar chave API > copia a chave gerada e volta no painel do Wordpress > menu do Mailchimp e cola a chave. 
No submenu > Formulário > pode configurar os campos que deseja colocar no formulário.
